# 项目目录分类

## 顶层目录说明

| 目录 | 作用 |
| --- | --- |
| `materials/` | 存放脱敏说明、公开参考资料占位；原始申报材料单独保存在仓库外 |
| `docs/` | 存放材料整理、目录说明、工作分工、方案草稿 |
| `configs/` | 模型配置、运行参数、接口示例配置 |
| `data/raw/` | 原始图像、原始文本、原始举证数据 |
| `data/interim/` | 中间处理产物，如抽取后的文本、临时 JSON |
| `data/processed/` | 结构化后的训练/评测数据 |
| `data/rules/` | 业务规则模板、字段定义、规则库 |
| `data/samples/` | 请求样例、Few-shot 样例、演示用输入 |
| `src/` | 项目主代码 |
| `tests/` | 冒烟测试与后续单元测试 |
| `outputs/` | 日志、评测报告、导出结果 |
| `examples/` | 示例文件或演示输入输出 |
| `scripts/` | 后续可放批处理脚本、数据清洗脚本 |

## `src/` 下的模块划分

| 目录 | 作用 |
| --- | --- |
| `src/core/` | 公共数据结构、基础类型定义 |
| `src/modules/rule_engine/` | 审查规则加载、规则组织 |
| `src/modules/model_gateway/` | 多模态模型接口封装 |
| `src/modules/prompt_engine/` | Prompt 模板与推理流程 |
| `src/modules/postprocess/` | 模型输出解析、结构化、修补 |
| `src/modules/evaluation/` | 评测指标、结果对比 |
| `src/app/` | 主程序入口、CLI 或 Demo |

## 推荐使用方式

### 申报与资料线

- 优先维护 `materials/` 和 `docs/`
- 用于脱敏说明、公开版文档、答辩材料草稿沉淀

### 数据与规则线

- 优先维护 `data/rules/`、`data/samples/`、`data/raw/`
- 用于准备图斑规则、样本、标准答案

### 开发与演示线

- 优先维护 `src/`、`tests/`、`outputs/`
- 用于写代码、跑 demo、做评测
